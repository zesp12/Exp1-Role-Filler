Overview of Exp 1

Participants complete a rapid serial image-detection task across two blocks:
- epoch in : participants memorize a "Containment" (_C) image and must detect it
- epoch on : participants memorize a "Support" (_S-top) image and must detect it

On each trial, participants see a fixation cross, then a blank screen, then the image, and must press:
- F if the image is the current block's target
- J if it is not
  
Sequence: intro -> language questionnaire -> demographics -> instructions (fullscreen, demo, reminder)
  -> epoch 1 intro -> epoch 1 trials (96)
  -> epoch 2 intro -> epoch 2 trials (96)
  -> send -> end 

Target Selection:
- One object pair is randomly chosen as the IN (Containment) target; a different pair is randomly chosen as the ON (Support) target.
- Epoch order (IN first vs. ON first) is randomized per participant.
- Each epoch's non-target pool is the full image set excluding the other epoch's target image (the current epoch's own target is filtered out separately when building non-targets).
- Within each sub-block: the epoch target appears 4 times, plus one copy of every remaining non-target image, all shuffled together.
- Each image is randomly assigned a horizontal-flip orientation per trial.
